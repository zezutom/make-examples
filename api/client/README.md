## How to run the examples

### Java
Each `Java` example is a standalone application, just run the `main` method.

### JavaScript
To explore JavaScript examples, please first run `node.js` server as follows:
```shell
cd ./nodejs
npm start app.js
```
Visit http://localhost:3000
